# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-26

### Added

- Initial release of Terminal Deck
- Multi-terminal grid layout support (2x2 to 4x4)
- Terminal group management
- Drag and drop terminals between groups
- Custom terminal labels (double-click to edit)
- Real-time terminal status indicators
- Keyboard shortcuts for common actions
- Cross-platform support (Windows, macOS, Linux)
- Persistent configuration storage
- Settings panel for customization
- Working directory selection per terminal
